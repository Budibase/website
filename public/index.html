Start to finish in 4 simple steps

Traditional development is time consuming - building web servers, writing boilerplate code, and wrestling with design takes weeks. Budibase removes the boilerplate work providing you with a platform to develop and design software 50x faster.

1. DATA

Load and manage your data with ease

With Budibase, you can connect to an external data source, import data, or start from scratch. You can then query, create views, and manage this data right within the Budibase GUI.

2. DESIGN

Build with pixel-perfect blocks

Budibase auto-generates beautiful screens, form your data, saving you precious dev time. It also comes with pixel-perfect, powerful components helping you build UIs users will love.

3. AUTOMATE

Automate manual processes

Save time by automating manual processes and workflows. From connecting to webhooks, to automating emails, simply tell Budibase what to do and let it work for you.

4. DEPLOY

Deploy your software with total control

In just a few simple clicks, you can deploy your software. You can host across different regions with Budibase (AWS) or self-host on your own infrastructure, the decision is yours.

FAQs

When is the launch?

We're in public beta and you can try it now for free. We will publicly launch in January/February 2021.

Do I need to know how to code

You do not need to know how to code, but in some cases, knowing some code will allow you to build better web apps.

Is Budibase open source?

Budibase is fair-code licensed - Apache 2.0 with Commons Clause

Can I build a website with Budibase?

Budibase is for building internal tools and custom software.

Is Budibase free?

Yes, to an extent. You will never pay for using the builder. There are certain processes you might need to pay for depending on the extent of your usage - for example, hosting and workflows.

Can I become a Budibase partner?

Not yet, but we are exploring the finer details of partnerships so please register your interest: Partnerships
