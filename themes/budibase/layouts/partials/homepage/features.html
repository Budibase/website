Here's how it works

Connect to data, or start from scratch

Connect to an external database, pull data from a Rest API, import a CSV, or start from scratch with Budibase's built-in database.

Supported databases

We support over 20 datsources, including PostgreSQL, MySQL, Oracledb, MongoDB, Airtable, Snowflake, MSSQL, Google Sheets, Rest APIs, and more.

No-code built-in database

Budibase comes with a built-in database that's built on top of CouchDB. It's incredibly easy to use and handy when you're building apps from scratch.

Instantly turn data into apps

Generate powerful CRUD apps and forms from your data with just a few clicks. No coding required!

Generate screens using SQL data

Yes, you can. Both MySQL and PostgreSQL are supported.

Public and private screens

With Budibase, you can have both private and public screens.

Build apps with premade blocks

Build apps people enjoy using with Budibase's premade blocks. From tables to charts, Budibase has everything you need to build apps in minutes.

30+ premade building blocks

Budibase comes with over 30 premade components and building blocks.

Create custom components

Build your own custom components, or source one from the community, and use them in your Budibase apps.

Design beautiful apps you're proud of

Create beautiful apps that work perfectly across mobile, tablet, and desktop. With Budibase's design features, design has never been easier!

Apps are 100% customizable

All Budibase components and apps are fully customizable. You can also add custom CSS to components.

Responsive by default

Budibase apps are fully responsive and work across all devices.

Automate workflows with ease

With just a few clicks, automate manual processes and streamline operations within your app and business.

Over 20 automation blocks

Choose from over 20 automation blocks, including CRON, webhooks, Slack notifications and more.

Run automations on a schedule

You can run your automations based on predefined or custom schedules.
